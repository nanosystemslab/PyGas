Alicat Devices

The Alicat devices module provides interfaces to control and interact with Alicat Mass Flow Meters and Pressure Transducers.

AlicatFlow Class

class pygas.AlicatFlow(port: str = '/dev/tty.usbserial-AU0585NK', baudrate: int = 19200)

Bases: object

Represents an interface to control Alicat flow meters.

The AlicatFlow class represents an interface for controlling Alicat flow meters. It provides methods to start and stop data streaming, poll data, tare the device, query available gases, change gas types, lock and unlock the device display, read streaming data, set units, query static unit value, set baud rates, and change unit IDs.

Usage Example

The following example demonstrates how to use the AlicatFlow and AlicatPressure classes to interact with the devices:

if __name__ == "__main__":
    alicat_flow = AlicatFlow()
    unit_id = input("Enter the current unit ID: ")
    alicat_flow.stop_stream(unit_id)
    gas_num = input("Enter the gas number: ")
    alicat_flow.change_gas(unit_id, gas_num)
    data = alicat_flow.poll_data(unit_id)
    print(data)
